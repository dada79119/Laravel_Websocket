# How to play
1. git clone https://github.com/dada79119/Laravel_Websocket.git
2. cp .env.example .env 
3. vi .env to connect your DB
4. php artisan key:generate
5. composer install
6. php artisan migrate
7. node socketserver/server.js
8. php artisan serve
